Activity Streams

a format for syndicating social activities around the web

Draft specs: Activities in Atom; Activity Schema; Atom Media.

Experimental specs: Schema, Atom, JSON

Join the mailing list. Contribute to the wiki, or fork us on GitHub.

The Activity Streams format has already been adopted by MySpace, Windows Live, Google Buzz, BBC, Opera, TypePad, Gowalla, Gnip, Socialcast, Superfeedr, YIID, and many others.

An initiative from the Diso Project. Background.

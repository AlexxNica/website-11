Audience Targeting for JSON Activity Streams

This spec is currently a draft. Its registry information is subject to change.

Activity Components

The following activity components are defined by this specification:

Name JSON Property Description
BCC bcc Array of Objects specifying the private secondary audience of an activity.
BTO bto Array of Objects specifying the private primary audience of an activity.
CC cc Array of Objects specifying the public secondary audience of an activity.
To to Array of Objects specifying the public primary audience of an activity.

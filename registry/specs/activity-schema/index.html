Activity Schema

This spec is currently a draft. Its registry information is subject to change.

Verbs

The following verbs are defined by this specification:

Name Identifier Description
Join join Becoming a member of the object
Make Friend make-friend Creating a friendship relationship that is reciprocated by the object
Mark as Favorite favorite Marking the object as an item of special interest
Mark as Liked like Indicating that one likes the object
Negative RSVP rsvp-no Indicating a negative RSVP for the object
Play play Spending some time enjoying the object
Positive RSVP rsvp-yes Indicating a positive RSVP for the object
Possible RSVP rsvp-maybe Indicating a possible RSVP for the object
Save save Calling out the object as being of interest primarily to the actor
Share share Calling out the object to attract others to it
Start Following follow Starting to follow the object
Tag tag Identifying the presence of the object inside the target
Update update Modifying the object

Object Types

The following object types are defined by this specification:

Name Identifier Description
Article article An article, such as a news article, knowledge base entry or similar construct
Audio audio An item of audio content
Bookmark bookmark A pointer to a URL
Comment comment A textual response to another object
Event event An event that occurs in a certain place during a particular interval of time
File file A document or other file with no additional machine-readable semantics
Folder folder A collection of files or media objects
Group group A social networking group
List list A collection of related objects
Note note A short-form text message
Person person A user account
Photo photo A graphical still image
Photo Album photo-album A collection of images
Place place A location on earth
Playlist playlist An ordered list of time-based media items
Product product A commercial good or service
Review review A primarily prosed-based commentary on another object
Service service Something that provides a service to others
Song song A song or a recording of a song
Status status A human-readable update of the author's situation, mood, location or other status
Video video An item of video content

Activity Components

The following activity components are defined by this specification:

Name JSON Property Description
Location An object construct representing the location at which the activity occured.
Mood A human-readable keyword or short phrase describing the mood of the actor at the time the activity was performed.

Object Components

The following object components are defined by this specification:

Name JSON Property Description
Embed Code embedCode A HTML fragment that provides a viewer or interactive player for the media asset of a media object
End Date and Time endTime For objects that cease to exist after a certain time, the time that the object ceases to exist
File URL fileUrl For an object that represents some kind of file, the URL at which the file can be retrieved
Full Content content A HTML fragment that represents the object excluding its title and author
Full Media Stream stream A media link construct linking to the full-fidelity media asset for a time-based media object
Icon icon A media link construct for a small icon representing the object
Larger Image fullImage A media link construct for an alternative version of the representative image that is larger and intended for standalone viewing
Latitude latitude For objects that have a physical presence on earth, the (possibly-approximate) latitude of its current location
Longitude longitude For objects that have a physical presence on earth, the (possibly-approximate) longitude of its current location
MIME Type mimeType For an object that has a File URL, an advisory MIME type for the resource at that URL
Rating rating For objects that represent some kind of opinion, a number representing the overall rating
Start Date and Time startTime For objects that exist only after a certain time, the time that the object begins to exist
Target URL targetUrl For an object that acts as a pointer to a URL, the URL that is the target of the pointer

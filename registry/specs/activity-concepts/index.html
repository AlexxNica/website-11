Activity Streams Concepts and Representations

This spec is currently a draft. Its registry information is subject to change.

Verbs

The following verbs are defined by this specification:

Name Identifier Description
Post post The act of authoring an object and then publishing it online.

Activity Components

The following activity components are defined by this specification:

Name JSON Property Description
Actor actor An object construct that describes who or what performed the action that this activity describes.
Body body An HTML representation of a natural language description of the activity.
Generator generator An object construct that represents the application that was used to perform the activity.
Icon iconUrl A URI of a small image that provides a visual representation of the activity.
Object object An object construct that describes the primary object of the activity.
Service Provider provider An object construct that represents the service that is publishing the activity.
Standard Link links A standard hypertext link whose source is the activity.
Target target An object construct that describes the object to which the activity was performed.
Time postedTime The time at which the activity occured.
Title title An HTML representation of the natural language title for the activity.
Verb verb A verb keyword that identifies the type of action performed.

Object Components

The following object components are defined by this specification:

Name JSON Property Description
Action Link actionLinks An action link construct representing an action that a human could perform on this object.
Attached Object attachedObjects An object construct that is attached to this object.
Downstream Duplicate ID downstreamDuplicates A string containing the ID value of another object construct which is a duplicate of this object created by re-publishing.
ID id An absolute URI that uniquely identifies the object.
In Reply To Object inReplyTo An object construct that this object is a response to.
Name displayName A human readable display name for the object, if the object has a name.
Object Type objectType An object type keyword that identifies the type of the object.
Permalink URL link The URI of a resource which provides an HTML representation of the object.
Reaction Activity reactions An activity construct that describes an activity whose object is this object.
Reply Object replies An object construct that has this object as its in reply to object.
Representative Image image A media link construct that identifies an image resource which provides a visual representation of the object.
Standard Link links A standard hypertext link whose source is the object.
Summary summary A human-readable description or summary of the object.
Upstream Duplicate ID upstreamDuplicates A string containing the ID value of another object construct for which this object is a duplicate created by re-publishing.

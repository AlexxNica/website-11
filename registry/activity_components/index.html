Activity Components

The following activity components are currently registered:

Name JSON Property Specification Description
Actor actor Activity Streams Concepts and Representations An object construct that describes who or what performed the action that this activity describes.
Body body Activity Streams Concepts and Representations An HTML representation of a natural language description of the activity.
Generator generator Activity Streams Concepts and Representations An object construct that represents the application that was used to perform the activity.
Icon iconUrl Activity Streams Concepts and Representations A URI of a small image that provides a visual representation of the activity.
Location Activity Schema An object construct representing the location at which the activity occured.
Mood Activity Schema A human-readable keyword or short phrase describing the mood of the actor at the time the activity was performed.
Object object Activity Streams Concepts and Representations An object construct that describes the primary object of the activity.
Service Provider provider Activity Streams Concepts and Representations An object construct that represents the service that is publishing the activity.
Standard Link links Activity Streams Concepts and Representations A standard hypertext link whose source is the activity.
Target target Activity Streams Concepts and Representations An object construct that describes the object to which the activity was performed.
Time postedTime Activity Streams Concepts and Representations The time at which the activity occured.
Title title Activity Streams Concepts and Representations An HTML representation of the natural language title for the activity.
Verb verb Activity Streams Concepts and Representations A verb keyword that identifies the type of action performed.
